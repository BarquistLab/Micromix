# Using Micromix

This is a breif guide for highlight the functionality of the Micromix website. 

The site was developed to be adaptable for use with multiple organisms and datasets that can be viewed through available plugins. New organisms/datasets can be added or removed, and plugins can be designed and added for additional visualisation options.


## Selecting organism

The first page allows you to select one of the available organisms/bacteria. 

<img width="80%" src="images/micromix_running.png" />

Once chosen, you have the option of selecting the desired datasets.

<img width="80%" src="images/select_dataset.png" />

You also have the choice of adding or removing specific columns.

<img width="50%" src="images/choose_additional_columns.png" />

Clicking **Add** will load the data
